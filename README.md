# GIT_DEMO

Some new docs.

- Clone the repository.
```bash
git clone <github-url>
```
- Make changes.
- Add changes to Git's history:
```bash
git status
git diff
git add <file-name>
git commit -m "<git-message>"
git log
```
- Send files to GitHub.
```bash
git push
```
